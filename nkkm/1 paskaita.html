Naudinga informacija

  • Dokumentacija

    • Unity dokumentaciją galite rasti čia. Dokumentaciją taip pat galima rasti:
    unity3d.com -> Learn -> Documentation -> Scripting API. Pagrindinis skyrius,
    kurio mums reikės yra UnityEngine -> Classes.

  • Skripto struktūra

    • Start(); - veikia tik vieną kartą.

    • Update(); - veikia kiekvieną kadrą (frame).

  • Jeigu neatsidaro skriptas

    • Jeigu atidarinėjant skriptą, Jums jo neatidaro ir rodo klaidą, tai reiškia, kad kodo editorius,
    su kuriuo norite atidaryti skriptą nėra suinstaliuotas, todėl mums reikės nurodyti kitą;,
    suinstaliuotą editorių;, mūsų atveju, Visual Studio 2017. Tai mes galime padaryti nueje i
    virsutini menu -> Edit -> Preferences... . Tada, atsidariusiame lange, keliaujame į External Tools
    ir pakeičiame "External Script Editor" nustatymą iš "Internal" į Visual Studio 2017. Jei pasirinkimuose nėra
    "Visual Studio 2017", tada teks surasti Visual Studio .exe paleidimo failą rankiniu būdu. Tokiu atveju renkamės "Browse..."
    ir atsidariusiame paieškos lange turime rasti kelią iki Visual studio .exe failo, pavadinimu "devenv.exe".
    Kelias iki šio failo turėtų būti kažkas panašaus į tai - "C:\Program Files (x86)\Microsoft Visual Studio\2017\Community\Common7\IDE\devenv.exe".
    Jei vis dar kyla nesklandumų, klauskite dėstytojo arba jo asistento!
