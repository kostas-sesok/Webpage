Naudinga informacija

  • Teksto kūrimas - su dešiniu pelės klavišu spaudžiam ant Hierarchy (tuščios vietos) -> UI -> Text.
  • Šriftai - juos galima atsisiųsti iš šios svetainės: www.dafont.com.
    1. Atsisiųskite norimą šriftą.
    2. Grįžkite į Unity. Assets -> (dešinį pelės klavišą) -> Import new Asset... Pridedam atsisiųstą šriftą.

    3. Pridedam jį (tempdami - "dragindami") į teksto Font langelį.
