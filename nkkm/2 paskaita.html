Naudinga informacija

  • Teksto kūrimas - su dešiniu pelės klavišu spaudžiam ant Hierarchy (tušios vietos) -> UI -> Text.
  • šriftai - juos galima atsisiūsti iš šios svetainės: www.dafont.com.
    1. Atsisiūskite norimą šriftą.
    2. Grįžkite į Unity. Assets -> (dešinį pelės klavišą) -> Import new Asset... Pridedam atsisiūstą šriftą.

    3. Pridedam jį (tempdami - "dragindami") į teksto Font langelį.
